Calculate and report RMSE and R-squared for each model. Keep in mind when calculating RMSE that the prediction and the observed outcome have to be on the same scale; if you compare a logged prediction (from a model with a log transformed outcome) with an unlogged outcome variable you will get a crazy answer that is an artifact of the mismatch in scaling. So, you will need to exponentiate the logged prediction (from the log-log model) in order to return it to the original scale and make it comparable to the unlogged outcome. By exponentiate I mean: raise e to the power of n, where n is the natural log of price. Remember that exponentiation undoes log transformation, and vice versa. Use this identity: \(e^{ln(x)}\) = x. In R this identity is exp(log(x)) = x.

This interaction model is a tricky model to interpret. Here is some guidance:

  1. Beverly Hills is the (missing) reference city in the model, assuming you have allowed the lm() function assign factor levels alphabetically.

  2. There are three sorts of coefficients reported in the regression output for an interaction model:

  • The intercept is the average value of the target when the inputs are 0 (for numeric variables) or the reference category (for categorical variables). Since there are two predictors in this model, sqft and city, the intercept will be average log price when log sqft is 0 and city is Beverly Hills. Log sqft will never be 0 so the intercept in this model is not interpretable.

  • The main effects are first four coefficients in the output. In the plot you created for this question you can see that there is a regression line for each city. Similarly in the model: there is no single relationship between log price and log sqft, and consequently the main effect for log sqftwhich is the first reported coefficient in the model outputis conditional on city. Specifically, it denotes the relationship between log sqft and log price in Beverly Hills, the reference city. The main effects for a predictor in an interaction model will always be conditional on the levels of the variable with which it has been interacted.

  • The interaction effects are the final 3 coefficients in the output. (The colon indicates the interaction, as in log(sqft):cityLong Beach.). These coefficients estimate the change in the slope of the regression line for each city compared to Beverly Hills. If the interaction coefficients are negative that means that the regression line relating log sqft to log price is less steep for that particular city in comparison to Beverly Hills, or, equivalently, that the relationship is less strong. Note that the coefficient for the first interaction term, log(sqft):cityLong Beach, is (very slightly) negative but not statistically significant. What does this mean? The slope of the regression line in LB is statistically indistinguishable from the one in BH.
